Департамент

Система обработки входящих файлов и задач

Общая информация

Данный Telegram-бот предназначен для автоматизации обработки входящих файлов, документов и задач в рамках рабочих процессов департамента.

Важно: Все файлы и сообщения обрабатываются в специальных темах (вкладках) внутри рабочей группы.

Основные темы (вкладки)

1
Входящие - сюда поступают все новые файлы и сообщения для обработки
2
Отработано - сюда перемещаются файлы после обработки
3
Архив - архивные материалы

Сценарий: Обработка нового файла

1
Загрузка файла
Отправьте файл, фото или текстовое сообщение в тему "Входящие" рабочей группы
2
Появление кнопок
Бот автоматически добавит сообщение с кнопками: Вне зоны и В работу
3
Выбор действия
Нажмите Вне зоны, если задача не относится к сфере ответственности департамента
Нажмите В работу, если задача требует обработки

Сценарий: Назначение ответственного

1
Выбор ответственного
После нажатия В работу появится список ответственных сотрудников
2
Назначение исполнителя
Выберите ответственного за выполнение задачи из списка
3
Выбор действия
Появятся кнопки: Отправить в работу и Доп. комментарий

Сценарий: Добавление комментария

1
Нажатие кнопки
Нажмите Доп. комментарий для добавления пояснений к задаче
2
Добавление пометки
К сообщению будет добавлена пометка "В работу (ФИО)"
3
Ввод комментария
Ответьте на сообщение бота с просьбой ввести комментарий
4
Перенос задачи
Сообщение с файлом/фото/текстом будет перенесено в тему "Отработано" с добавленным комментарием

Сценарий: Отправка в работу без комментария

1
Нажатие кнопки
Нажмите Отправить в работу для немедленной отправки задачи
2
Перенос задачи
Сообщение будет перенесено в тему "Отработано" с пометкой "В работу (ФИО)"

Правила работы с ботом

Для эффективной работы следуйте этим правилам:

•
Всегда отправляйте файлы и сообщения в тему "Входящие"
•
Назначайте ответственных только из предоставленного списка
•
При добавлении комментариев обязательно отвечайте на сообщение бота
•
Не отправляйте случайные сообщения в темы бота
